## 🔧 技术架构

### 前端技术栈
- **框架**: Flutter 3.x
- **状态管理**: GetX
- **UI设计**: Material Design 3
- **导航**: GetX Navigation

### 后端技术栈
- **数据库**: PostgreSQL (Supabase)
- **认证**: Supabase Auth
- **实时数据**: Supabase Realtime
- **存储**: Supabase Storage

### 架构模式
- **分层架构**: Presentation → Business Logic → Data Access
- **服务层**: 业务逻辑封装
- **控制器层**: 状态管理和UI交互
- **数据层**: 数据库操作和API调用
